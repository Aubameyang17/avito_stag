Обязательно использовать уникальный id в диапазоне 111111-999999

Test1 - Создать объявление (ожидаем положительный результат)
Цель: Проверить, что объявление создается
Условия: Сервер доступен, уникальный sellerID
Что делать:
1. Сгенерировать уникальный sellerID
2. Выполнить `POST /api/1/item` с JSON телом:
{
  "sellerID": 111111,
  "name": "test item",
  "price": 100,
  "statistics": {"likes": 1, "viewCount": 1, "contacts": 1}
}
Поля likes, viewCount, contacts - обязательны
3. Проверить код ответа: должен быть 200
4. Проверить тело ответа: статус возвращается строкой с id

Test2 - Получить объявления по его идентификатору (ожидаем положительный результат)
